# Security Policy

## Project status

This repository is an educational project and is not yet production-ready.

## Reporting a vulnerability

Do not publish credentials, tokens, passwords, private keys, or personal data in a public issue.

Report security concerns privately to the repository owner.

## Current security limitations

- Production authentication is not completed.
- Production authorization hardening is not completed.
- Rate limiting is not completed.
- Formal penetration testing has not been performed.
