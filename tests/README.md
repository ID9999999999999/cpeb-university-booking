# Tests

This directory classifies verification by test level.

## Taxonomy

- `unit/` — isolated classes and functions.
- `integration/` — database and module interactions.
- `e2e/` — complete user and API flows.
- `performance/` — load and latency verification.
- `security/` — security-focused verification.
