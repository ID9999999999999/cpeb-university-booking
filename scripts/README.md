# Scripts

This directory is reserved for repository-level automation.

## Taxonomy

- `development/` — local developer utilities.
- `database/` — database setup and maintenance.
- `verification/` — repeatable checks and evidence generation.
- `release/` — packaging and release automation.

## Current status

Backend functional verification scripts currently remain under `apps/api/scripts/`
because they directly belong to the backend application.

Future repository-wide scripts will be classified here.
